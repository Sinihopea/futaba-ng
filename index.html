futaba.php

futaba.php is Message board scripts.

I branched script to this repository. Is script used in http://www.2chan.net/.

This script aims running with PHP 5.4.15 or later...

How to running

  • PHP 5.4.15 or later
> cd app
> grunt concat:model
> grunt concat:futaba
> grunt shell:phpRunning
